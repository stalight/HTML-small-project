Some difficulties I've faced during my time programming the websites

Problem #1:

First I had some problems letting the computer detect whether or not the box its ticking has already been ticked or not by the player and the bot (which causes it to replace the tick that the player inputted)

Solution:

I used a loop to solve the problem where the value of the box that’s not ticked is the same and the box that is ticked are different.
(For example: Box that are not played = 0 and Box that are played = 1) _______________________________________________________________________________________________________________

Problem #2:

Then I had trouble making a tie happen because the computer will get into an infinite loop on the last click from the player because the program is set that the computer will always go after the player and because the computer cannot find the tile to mark its own tile after the last tile is taken, it goes into a infinite loop.

Solution:

I solved it by changing the order in which the function activates in as well as its conditions, for example the computer will only go if the previous function is true for one of the conditions and that is if the player didn't take the last tile. _______________________________________________________________________________________________________________

Problem #3:

Giving Divs value is not always working for project 3 because I'm too dependent on it due to the effectiveness of it when I was working with it.
